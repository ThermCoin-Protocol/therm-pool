## Mining Quick-Start Guide

1. **Download Thermcoin Executable**: [ThermCoin GitHub](https://github.com/ThermCoin-Protocol/go-thermcoin/releases/)
2. **Unzip and Setup**: Place "therm" executable in folder with genesis block (genesis.json).
3. **Data Directory Setup**:
   ```shell
   mkdir myNode
   therm init genesis.json
   ```
4. **Start Mining**:
   ```shell
   therm --bootnodes <bootnode-enode-url> --nodiscover --snapshot=false --mine
   ```

5. **Bootnodes**:
